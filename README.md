# Showdown to Radical Cobblemon Trainer Data Converter

A simple tool to convert Pokemon Showdown team data into Radical Cobblemon trainer data.

## Versions as of last update

- Pokemon: Generation 9
- Radical Cobblemon Trainers: 0.18
- Cobblemon: 1.7.3
- Mega Showdown: 1.9.1

## Note

This tool is not perfect, I'm not using Cobblemon data to validate the results (especially since data packs could change what is and isn't a valid species, ability, move, etc). If you do run into issues, please submit an issue.

If you run into edge cases while using this tool (for example, in Cobblemon Arcanine-hisui is actually Arcanine with the hisuian aspect), please submit an issue so that I can add the proper correction to the corrections file. Or if you have the knowledge, create a pull request with the correction added to the correction.json file.
